# Проверка сбалансированности скобок

Реализация проверки сбалансированности скобочной последовательности с использованием стека.

## Описание

Программа определяет, является ли введённая последовательность скобок `()`, `{}`, `[]` сбалансированной. Для этого используется структура данных **стек**, реализованная на основе списка.

## Использование

1. Запустите скрипт:
`bash python stack.py`
2. Введите последовательность скобок, например:
`({[}])`
3. Программа выведет результат проверки, например:
`({[]})` -> Сбалансированно


## Примеры

- `()` -> Сбалансированно  
  - `({[]})` -> Сбалансированно  
- `((()` -> Несбалансированно  
- `([)]` -> Несбалансированно

## Примечания

  - Поддерживаются скобки всех трех типов: `()`, `{}`, `[]`
  - Проект выполнен в рамках обучения.
